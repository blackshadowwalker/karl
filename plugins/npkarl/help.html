This the the plugin:

This NPAPI application for firefox/chrome.

Setting:

  • Chrome:
    Get Chrome Path(such as : D:\AppData\Local\Google\Chrome\Application\25.0.1364.84\Extensions). 
Edit file external_extensions.json.
Add the follow code :
{
  "description": "karl desp", 
  "name": "karl extension",
  "plugins": [
    { "path": "E:/karl/Teleframe/src_code/Karl_code/npkarl/Debug/npkarl.dll", "public": true }
  ]
}
  • Firefox: 
    Edit regedit.
Find [HKEY_CURRENT_USER\Software\MozillaPlugins\] or [HKEY_LOCAL_MACHINE\SOFTWARE\MozillaPlugins\]
Add subkey (named such "karl plugin")
Add	key named "path" @ string type.
The value of key[path] is the path of your np*.dll.
  such as:
	"path"="E:\\karl\\Teleframe\\src_code\\Karl_code\\npkarl\\Debug\\npkarl.dll"
  • IE: Please Develop ActiveX/ocx.
